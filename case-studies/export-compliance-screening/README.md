# Export Compliance Screening Process Improvement Case Study

This folder contains all artifacts related to the Export Compliance Screening Process Improvement project.

## Project Overview

This case study demonstrates an end-to-end Business Analysis engagement for improving the Export Compliance Screening process within a manufacturing organization using ERP software.

The objective of the project is to analyze the current manual compliance screening process, identify operational inefficiencies, gather business requirements, and recommend improvements while ensuring compliance with international export regulations.

> **Note:** This is a fictional project inspired by real-world Global Trade Compliance processes. It is created for learning and portfolio purposes only.

---

# Business Problem

The organization processes approximately **4,000 international export orders per month**.

As export volumes continue to increase, the existing manual compliance screening process has become inefficient.

The current process results in:

- Manual compliance reviews for every international order
- Increasing review backlogs
- Shipment delays
- Inconsistent audit documentation
- Heavy analyst workload
- Limited visibility into operational performance

The business engaged a Business Analyst to understand the current process, identify root causes, and recommend improvements.
